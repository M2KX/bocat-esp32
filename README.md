# bongo cat with esp32

this is a `ESP32` code to draw and control a bongo cat hands.

---

## Parts 

- Generic ESP32
- 0.96in OLED Screen
- Push Buttons X 2
- Wires
- Perfboard (Optional)
- Enclosure

---

## Wiring 

#### OLED

GND --- Board Ground

VCC --- Board 3.3v

SCK --- GPIO22(D22)

SDA --- GPIO21(D21)

#### Buttons

Button1 --- GPIO14(14)

Button1 --- GPIO13(D13)

##### both buttons are connected to 3.3v pin rather than ground

##### refer to the diagram for more information
